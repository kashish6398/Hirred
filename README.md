# 🚀 Hirred – Job Portal Platform

A modern and scalable **Job Portal Web Application** designed to connect job seekers with recruiters seamlessly.
This platform enables users to explore jobs, save opportunities, and manage job postings with an intuitive and responsive interface.

---

## 🌐 Live Demo

🔗 https://hirred-sepia.vercel.app/
🔗 https://hirredportal.netlify.app/

---

## ✨ Key Features

### 👤 Authentication

* Secure user authentication
* Protected routes for authorized access

### 💼 Job Exploration

* Browse available job listings
* View detailed job descriptions
* Search and filter jobs

### ⭐ Saved Jobs

* Save jobs for later access
* Manage your saved job list easily

### 🧑‍💻 Recruiter Panel

* Post new job opportunities
* Manage your job listings
* View all jobs created by you

### 🎨 User Experience

* Fully responsive design
* Clean and modern UI
* Smooth navigation across pages
* Dark theme

## ⚙️ Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/hirred.git
cd hirred
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the project

```bash
npm run dev
```
